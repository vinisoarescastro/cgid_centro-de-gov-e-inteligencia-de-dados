# PortalBI — Portal Corporativo de Inteligência Analítica

Portal web para centralizar relatórios Power BI Embedded com controle granular de permissões, auditoria completa e gestão de acesso corporativo.

## Início Rápido

```bash
docker-compose up
```

## Estrutura do Repositório

- `frontend/`   → Aplicação React (SPA)
- `backend/`    → API
- `docs/`       → Documentação técnica e de produto
- `prototipo/`  → Protótipo visual de referência (portal_v4_8.html)
- `scripts/`    → Scripts de automação

## Documentação

Acesse `docs/README.md` para o índice completo da documentação.

## Pré-requisitos

- Node.js 20 LTS
- Docker + Docker Compose
- Conta Azure com App Registration (Power BI Embedded)
